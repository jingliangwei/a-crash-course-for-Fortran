十二、语句函数、函数子程序和子例行程序

返回首页

语句函数

语句函数的使用如下：

f(x)=x**2-1 g(x)=2*x write(*,*) f(12),g(i)

函数子程序

函数子程序的使用如下：

program my_add integer a,b,sum read(*,*) a,b sum=my_add(a,b) write(*,*) sum end program function my_add(x,y) integer my_add integer x,y my_add=x+y end function

子例行程序

子例行程序的使用如下：

program my_add integer a,b,sum read(*,*) a,b call my_add(a,b,sum) write(*,*) sum end program subroutine my_add(x,y,ans) integer x,y,ans ans=x+y end subroutine
